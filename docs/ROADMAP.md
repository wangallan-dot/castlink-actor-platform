# 开发路线图

## 产品范围

当前阶段只服务两个身份：

- 演员
- 剧组方

核心内容对象只有：

- 演员资料
- 剧组招募

核心行为为：浏览、收藏、申请或邀请、沟通、试镜。当前阶段完全不考虑会员、套餐、广告、付费曝光和其他商业化能力。
