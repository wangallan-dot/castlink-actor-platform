# 幕见 CastLink 产品说明

## 1. 产品定义

幕见是连接演员、经纪公司和影视项目方的数字演员库与在线选角协作平台。

对演员，它提供可持续更新的职业身份、角色机会和试镜管理；对项目方，它提供演员搜索、候选人管理、试镜收集和团队评审；对经纪公司，它最终提供演员、档期和项目关系管理。

## 2. 核心用户

### 演员

- 创建和维护演员卡
- 上传形象照片、介绍视频和 Showreel
- 管理履历、技能、方言和档期
- 浏览角色并提交资料
- 接收试镜邀请
- 查看曝光、收藏和申请状态

### 项目方

- 创建项目与角色
- 搜索和筛选演员
- 建立候选演员名单
- 发送试镜任务
- 收集 Self-tape
- 团队评分、评论和定角

### 经纪公司

- 批量维护旗下演员资料
- 管理档期与项目推荐
- 管理经纪人子账号和权限
- 建立机构私有演员库

## 3. MVP 核心闭环

```text
演员注册并建卡
→ 项目方搜索演员
→ 收藏或发送角色邀请
→ 演员查看角色并提交资料
→ 项目方将演员加入候选看板
→ 邀请试镜并完成评审
```

## 5. 商业模式

### 主要收入

1. 项目方 SaaS 订阅
2. 经纪公司 SaaS 订阅
3. 演员专业会员
4. 项目认证与紧急选角服务
5. 后期电子合同、结算和保险等生态服务

### 收费原则

- 演员基础资料与接收邀请永久免费；
- 项目方为高级搜索、联系次数和协作流程付费；
- 不以付费购买演员自然搜索排名作为核心机制；
- 不向演员承诺角色或进组结果；
- 早期不抽取演员片酬。

## 6. 平台治理

- 项目方身份认证
- 项目信息审核
- 报酬与拍摄时间必填
- 诈骗和骚扰举报
- 联系方式分级开放
- 未成年人演员监护机制
- 保密项目水印和访问记录
- 演员资料下载权限控制
