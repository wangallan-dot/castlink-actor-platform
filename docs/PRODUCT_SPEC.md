# 幕见 CastLink 产品说明

## 1. 产品定义

幕见是面向演员和剧组方的移动端剧组招募与演员发现平台。

演员通过地区、招募时限和人物条件浏览真实剧组招募；剧组方通过地区、档期和形象条件浏览演员。双方围绕具体剧组招募建立沟通和试镜流程。

## 2. 当前身份

### 演员

- 建立和维护演员资料
- 设置常驻城市和可工作地区
- 浏览剧组招募
- 收藏和申请招募
- 接收剧组邀请
- 沟通并提交试镜

### 剧组方

- 发布和管理剧组招募
- 设置拍摄地区、日期、截止时间和报酬
- 浏览演员
- 邀请演员
- 接收演员资料
- 沟通并管理试镜

## 3. 核心闭环

```text
剧组发布招募
→ 演员浏览并提交资料，或剧组浏览演员并发出邀请
→ 对方接受
→ 自动建立绑定招募和演员的会话
→ 剧组发送试镜任务
→ 演员提交 Self-tape
→ 剧组评审并更新结果
```

## 4. 移动端信息架构

### 顶部频道

- 推荐
- 剧组招募
- 演员

### 地区导航

地区是两个内容模块的共同筛选条件：

- 剧组招募：实际拍摄地区
- 演员：常驻地区或可工作地区

### 演员端底部导航

- 首页
- 剧组招募
- 消息
- 申请
- 我的

### 剧组端底部导航

- 首页
- 演员
- 消息
- 招募管理
- 我的

## 5. 剧组招募时限

- 7天以上：正常招募
- 3至7天：招募进行中
- 24至72小时：即将截止
- 24小时内：今日截止或急招
- 超过截止时间：已截止，禁止继续申请

## 6. 平台原则

- 不允许脱离具体招募的大规模陌生私聊
- 报酬、拍摄地区、日期和截止时间必须明确
- 演员联系方式分级开放
- 剧组身份和招募信息需要审核
- 当前阶段不设计任何商业化、会员或付费曝光功能
