- 增加全部、未读、待回复、试镜和定角筛选
- 将系统截止提醒与双方聊天分开呈现
- 新增手机端会话列表与聊天详情切换
- 新增桌面端双栏消息布局

- 聊天顶部固定显示项目、角色和当前阶段
- 版本升级至 `0.3.0`
- PWA 缓存升级为 `castlink-mobile-v3`

### 验证结果

- Oxlint：0 warnings，0 errors
- TypeScript：通过
- Vite production build：通过

### 当前限制

- 消息为前端原型数据
- 发送内容仅保存在当前页面状态中

### 下一开发项
